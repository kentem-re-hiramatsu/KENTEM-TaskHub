# DatePicker

カレンダーUI で日付を選択するピッカーコンポーネント。

## 概要

月ごとのカレンダーグリッド。前後の月への移動、年月セレクター統合。本日マーク、選択日のハイライト。

## Props

| 名称 | 型 | 必須 | デフォルト | 説明 |
|------|------|------|------|------|
| `selectedDate` | `Date` | | - | 選択中の日付 |
| `onSelect` | `(date: Date) => void` | ○ | - | 日付選択時のコールバック |
| `size` | `"small" \| "medium" \| "large"` | | `"small"` | ピッカーのサイズ |
| `shouldCenterYear` | `boolean` | | `false` | 年セレクターを中央配置 |

詳細はスキル定義またはソースコードを参照してください。
