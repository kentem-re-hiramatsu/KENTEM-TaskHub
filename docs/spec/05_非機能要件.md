## 5. 非機能要件

| 項目 | 要件 |
|------|------|
| 対応ブラウザ | Chrome / Edge 最新版 |
| レスポンシブ | PC メインを想定。タブレット対応は Nice to have |
| 認証 | **必須**。GitHub OAuth ログインのみ許可し、`@kentem` 所属ユーザー以外はアクセス拒否 |
| データ保存 | サーバーサイド DB（詳細は設計フェーズで決定）|
| API 同期レスポンス時間 | 目安：1 プロジェクト 5 秒以内（タスク数 500 件想定）|
| CSV ファイルサイズ上限 | 10MB 以下（目安。サブ機能のため緩めの上限）|
