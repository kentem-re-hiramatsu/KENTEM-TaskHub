## ユーザー別履歴（メンバー詳細）

### 概要
プロジェクト詳細のメンバー一覧から遷移する画面。現在および過去の所属PJごとに、イテレーション別の実績・稼働情報を表示する。

### 遷移元
- 設定画面のメンバー一覧

### 表示項目
| 項目 | 説明 |
|------|------|
| 所属PJ一覧 | 現在および過去に所属していたプロジェクトをすべて表示 |
| 区分 | FE / BE / テスト |
| 目標ベロシティ | 計画ベロシティ（PL が設定した値）|
| 実績ベロシティ | 消化ポイント ÷ 稼働日数 |
| 開発効率 | 実施時間合計 ÷ 予定時間合計 |

### 操作
- **イテレーション切替**：過去・現在のイテレーションを選択可能（次イテレーション以降は選択不可）
- **集計範囲切替**：「選択のみ」（選択イテレーション単体）/ 「選択まで累計」（開始〜選択イテレーションの集計）

### 権限
- `admin` / `superAdmin` のみ閲覧可能
- URL 直アクセスも制限する（`user` は権限不足としてリダイレクト）

### 補足
詳細仕様は [project_progress_spec.md](../../project_progress_spec.md) の §2.8 を参照。
