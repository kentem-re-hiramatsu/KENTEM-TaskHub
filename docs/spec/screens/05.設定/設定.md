# 設定

## 概要

イテレーションカレンダー管理を提供する管理者向け設定画面。

**アクセス経路：** サイドバー「設定」

---

## 権限

| 操作 | 必要権限 |
|------|---------|
| イテレーションカレンダー編集 | `superAdmin` / `admin` のみ |

---

## イテレーションカレンダー設定

イテレーション（期）ごとの開始日・終了日・稼働日を管理する。

### 表示項目

| 項目 | 説明 |
|------|------|
| 開始日 | イテレーションの開始日 |
| 終了日 | イテレーションの終了日 |
| 稼働日 | 土日を除いた稼働日数（自動計算） |

- 期はドロップダウンで切り替え可能
