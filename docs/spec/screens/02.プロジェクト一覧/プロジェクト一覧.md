# プロジェクト一覧

## 概要

全プロジェクトの進捗カードを一覧表示する横断ビュー。カードクリックでプロジェクト詳細に遷移する。

**アクセス経路：** サイドバー「プロジェクト一覧」

---

## 権限

| ロール | カード一覧 | 遷移アイコン | プロジェクト詳細 |
|--------|-----------|------------|----------------|
| `user` | 全PJ表示 | 所属PJのみ表示 | 所属PJのみ遷移可 |
| `admin` / `superAdmin` | 全PJ表示 | 全PJ表示 | 全PJ遷移可 |

`user` が未所属プロジェクトへ直接 URL 遷移した場合はアクセス拒否とする。

---

## 主な表示要素

- 進捗カード一覧（全プロジェクト）
- 各カードの同期ボタン
- PJカードの表示/非表示切り替え

---

## 進捗カード一覧

各プロジェクトごとにカードを表示する。

| 表示項目 | 説明 |
|---------|------|
| プロジェクト名 | カードヘッダー左 |
| 詳細画面へのアイコンボタン | カード右上に配置。詳細画面へ遷移。`user` は未所属PJのカードでは非表示 |
| バージョン | プロジェクトのバージョン（例：Ver 1.8.0）|
| 現在のイテレーション | 現在進行中のイテレーション番号（例：11 イテレーション）|
| 進捗バー | 実績ポイント / 目標ポイントの割合（%）|
| 予定進捗 | 経過日数から算出した理想進捗率 |
| 実績進捗 | 完了ポイント / 目標ポイント |

---

## PJカード 表示/非表示

ユーザーは一覧に表示するPJをカスタマイズできる。

### 操作フロー

| 操作 | 動作 |
|------|------|
| 編集ボタンを押す | 各カードに「×」ボタンが表示される |
| カードの「×」を押す | そのPJのカードが非表示になる |
| 追加ボタンを押す | 非表示中のPJを選択するダイアログが開く |
| ダイアログでPJを選択 | 選択したPJのカードが再表示される |

### 状態の保存

非表示にしたPJの設定は **localStorage** に保存する（ブラウザ単位で保持）。

---

## プロジェクト詳細（カードクリック時）

カードをクリックするとプロジェクト詳細画面に遷移する。

閲覧権限は [権限](#権限) セクションを参照。  
`user` が未所属プロジェクトへ直接 URL 遷移した場合もアクセス拒否とする。

### 表示内容

担当プロジェクト画面と同一のレイアウト・機能を提供する。  
→ 詳細は [担当プロジェクト.md](./担当プロジェクト.md) を参照。
