# メンバー

## 概要

登録済みメンバーの一覧表示・登録・編集を提供する管理者向け画面。

**アクセス経路：** サイドバー「メンバー」

---

## 権限

| 操作 | 必要権限 |
|------|---------|
| メンバー一覧の閲覧 | `superAdmin` / `admin` のみ |
| メンバー登録・編集 | `superAdmin` / `admin` のみ |

---

## メンバー一覧

登録済みメンバーの一覧を表示する。

### 表示項目

| 項目 | 説明 |
|------|------|
| GitHub Account Name | GitHub のアカウント名 |
| 表示名 | アプリ内で使用する名前 |
| GitHub Account ID | GitHub アカウント識別子 |
| 権限 | メンバーの権限（`superAdmin` / `admin` / `user`） |

- 一覧からメンバーの登録・編集が可能

---

## メンバー登録・編集

手動でメンバー情報を新規登録・編集する。

### 入力項目

| 項目 | 説明 | 編集可否 |
|------|------|----------|
| 表示名 | アプリ内で使用する名前 | 常に編集可 |
| GitHub Account Name | GitHub のアカウント名 | 追加・編集不可 |
| GitHub Account ID | GitHub アカウント識別子 | TaskHub ユーザーと未紐づけの場合のみ編集可 |

### 補足

- メンバーは初回ログイン時に GitHub Account ID の一致で自動設定される
- TaskHub ユーザーと GitHub Account ID が紐づいた後は、GitHub Account ID の変更は不可
- 登録・編集操作は `superAdmin` / `admin` のみ実施可能
