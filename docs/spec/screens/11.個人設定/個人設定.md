# 個人設定

## 概要

ログインユーザー自身のアカウント情報を確認・編集する。

**アクセス経路：** ヘッダーメニュー「個人設定」（`/profile`）

---

## 権限

- 全ロール（`user` / `admin` / `superAdmin`）が自分自身の情報を閲覧・編集可能
- URL 直アクセスも制限する（未ログイン状態はログイン画面へリダイレクト）

---

## 表示項目

| 項目 | 表示 | 編集可否 | 説明 |
|------|------|----------|------|
| GitHub Account Name | 必須 | 不可 | GitHub ログイン時に取得したアカウント名 |
| 表示名 | 必須 | **可** | アプリ内表示用の名前。ユーザー本人が変更可能 |
| GitHub Account ID | 必須 | 不可 | GitHub アカウント識別子 |
| 権限 | 必須 | 不可 | `superAdmin` / `admin` / `user` のいずれかを表示 |

表示名のみ編集可能とし、他項目は参照専用とする。
