# プロジェクト詳細

## 概要

選択したプロジェクトの詳細を表示する。

**アクセス経路：** プロジェクト一覧・担当プロジェクト一覧などからプロジェクトを選択

---

## 権限

| ロール | カード一覧 | プロジェクト詳細 |
|--------|-----------|----------------|
| `user` | 全PJ表示 | 所属PJのみ遷移可 |
| `admin` / `superAdmin` | 全PJ表示 | 全PJ遷移可 |

---

## 画面構成

画面は「固定ヘッダー」と「タブコンテンツ」に分かれる。

```
┌────────────────────────────────────────────┐
│  固定ヘッダー（常時表示）                   │
│  - 進捗カード（全体 / FE / BE / テスト）    │
│  - イテレーション切替                       │
│  - 進捗集計切替                             │
│  - 同期ボタン・最終同期時刻                 │
├────────────────────────────────────────────┤
│  [ 概要 ] [ タスク ] [ 稼働日管理 ]         │
├────────────────────────────────────────────┤
│  タブコンテンツ（タブに応じて切替）         │
└────────────────────────────────────────────┘
```

---

## 固定ヘッダー

どのタブを選択していても常に表示される。

### 進捗カード

各区分（全体 / FE / BE / テスト）ごとにカードを表示する。

```
┌─────────────────────────────────────────────┐
│  Project A  [全体]              2026 IT#3   │
│  進捗  65%  [██████████░░░░░]               │
│  予定 70%   実績 65%                        │
│  完了: 13pt / 目標: 20pt                    │
└─────────────────────────────────────────────┘
```

| 表示項目 | 説明 |
|---------|------|
| 進捗バー | 実績ポイント / 目標ポイントの割合（%）|
| 予定 % | 経過日数から算出した理想進捗率 |
| 実績 % | 完了ポイント / 目標ポイント |
| 完了 pt / 目標 pt | 数値表示 |

### イテレーション切替

過去イテレーションと現在イテレーションのみ選択可。次イテレーション以降は選択不可（無効表示）。

### 進捗集計切替

カードの集計範囲をイテレーション切替とは独立して切り替え可能。

| モード | 内容 |
|--------|------|
| 選択のみ | 選択中イテレーション単体の値 |
| 選択まで累計 | 開始イテレーションから選択中まで集計した値 |

切替対象は「過去イテレーション」と「現在イテレーション」のみ。

### 設定リンク

ヘッダー右上に「設定」リンクを表示する。クリックすると `/projects/[id]/settings`（プロジェクト設定画面）へ遷移する。

| 権限 | 表示 |
|------|------|
| `admin` / `superAdmin` | 表示 |
| `user` | 非表示 |

### 同期ボタン・最終同期時刻

データの手動同期を実行するボタンと、最終同期時刻を表示する。

## タブコンテンツ

### 概要タブ

メンバーの状況とプロジェクト全体の進捗推移を確認できる。

#### メンバー一覧

`メンバー名 / 区分 / 稼働日数 / 計画ベロシティ / 実績ベロシティ / 開発効率 / 目標ポイント` を表示。

| 指標 | 定義 |
|------|------|
| 実績ベロシティ | 完了ポイント ÷ 稼働日数 |
| 開発効率 | 実施時間合計 ÷ 予定時間合計（1.0 が予定通り。1.0 超で超過）|
| 目標ポイント | 稼働日数 × 計画ベロシティ（手動上書き可）|

イテレーション切替で表示対象が変わる。

#### 進捗推移・予想グラフ

折れ線グラフで以下の 3 本の線を表示する。

| 線の種類 | 説明 | 表示区間 | 線種 |
|---------|------|---------|------|
| 実績進捗率 | 各イテレーション終了時点の実際の進捗率 | 開始〜現在イテレーション | 実線 |
| 理想進捗率 | イテレーション数に応じた均等な理想ライン | **全イテレーション（開始〜最終）** | 実線 |
| 予想進捗率 | 全イテレーションの平均ベロシティをもとに算出した予測ライン | 現在イテレーション〜将来 | 点線 |

**グラフ表示ルール**

| ルール | 詳細 |
|--------|------|
| X 軸 | プロジェクトの全イテレーション（開始〜最終）を表示する |
| 理想（予定）線 | 全イテレーション分を引く |
| 実績線 | 現在イテレーションまで引く（未来は表示しない）|
| 予想線 | 現在イテレーションの**実績値**を起点として引く（実績線と接続）|

**予想進捗の計算方法**

```
平均ベロシティ = 全イテレーション（過去〜現在）の完了ポイント合計 ÷ イテレーション数
予想完了ポイント = 現在の完了ポイント + 平均ベロシティ × 残イテレーション数
予想進捗率 = 予想完了ポイント ÷ 目標ポイント合計
```

### タスクタブ

プロジェクトのタスク一覧を表示する。

### 稼働日管理タブ

メンバー × イテレーション単位で稼働日数を管理する。

#### デフォルト稼働日

システム全体で「イテレーションあたりデフォルト稼働日数」を固定設定する（例：2週間イテレーションで祝日なし → 10日）。全プロジェクト・全メンバー共通で適用される。

#### 個別変更（メンバー × イテレーション単位）

有給取得・アサイン率変更・病欠などに対応して個別変更できる。

| 項目 | 説明 |
|------|------|
| 稼働日数 | デフォルト値を初期表示。手動上書き可 |
| 変更フラグ | デフォルトから変更された場合、値を**赤字**で表示 |
| 変更メモ | 変更理由を自由記述（例：「有給3日取得」）|

```
イテレーション #3（2026/04/01〜2026/04/14）  デフォルト稼働日: 10日

┌──────────┬──────┬──────────┬──────────────────────────┐
│ メンバー │ 区分 │ 稼働日数 │ 変更メモ                 │
├──────────┼──────┼──────────┼──────────────────────────┤
│ 田中     │ FE   │ 10       │                          │
│ 鈴木     │ FE   │ 🔴 7     │ 有給3日取得              │
│ 佐藤     │ BE   │ 10       │                          │
│ 山田     │ BE   │ 🔴 5     │ 他PJアサイン50%のため    │
└──────────┴──────┴──────────┴──────────────────────────┘
```

過去イテレーションの稼働日・メモも参照可能（読み取り専用）。
